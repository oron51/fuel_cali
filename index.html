RX gauge pairing
TX address
Connect to RX (the FUELRX unit) — it shows the TX address it's currently bound to. To change it, type the TX's 12-hex address and tap Bind to RX. The RX then drives the needle from that TX's broadcast.
Connection password
Optional. Locks the TX with a password + binding so only you can reconnect to recalibrate. Applied when you tap Start Beacon. This protects the calibration connection only — a scanning RX still reads the broadcast level without it.
Beacon update interval60 s
How often the in-tank unit wakes, measures, and broadcasts. Longer = longer battery life, slower updates. 2 s is handy for bench testing; 60–120 s for normal use.
